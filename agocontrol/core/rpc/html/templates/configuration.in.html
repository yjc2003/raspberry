A scenario is a set of multiple commands which can fired by a single command. For example you can create a scenario like "go to bed" which turns off your TV, switch off lights, close drapes and arm your security system.


Open Scenario Editor

You can fire a command or even a scenario if some event occours. For example you can create a event that is beeing triggerd at sunrise and fires a command to open the drapes. You can react on any event which is beeing sent by the system. Events also can be nested if you need some event like "if daytime" and "sensor brightness less 30" and "motion sensor trapped" then turn on the outdoor-light.


Open Event Editor
